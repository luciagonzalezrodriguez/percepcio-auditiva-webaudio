# percepcio-auditiva-webaudio

 Aural training exercices by detecting pitch with a YIN-Algorithm and using the WebAudio and VexFlow libraries. 
 4 Exercices are inclouded: 
 - Frequency tuning
 - Interval singing
 - Atonal chord reproducing
 - Atonal sequences singing

Supported by the followng browsers: Opera, Safari

You can find the web to practice at: luciagonzalezrodriguez.github.io/percepcio-auditiva-webaudio
